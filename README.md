Le nom du fichier est : `.streamlit/config.toml`

Cela signifie qu'on crée un **dossier** `.streamlit` avec un fichier `config.toml` dedans.

### Comment faire sur GitHub :

1. Cliquez sur **"Add file"** → **"Create new file"**
